# HEI_RN

## 설치 및 실행

### 패키지 설치

```bash
yarn
```

### 개발환경 실행

```bash
yarn start
```

## 구성 및 선정 이유

- 환경 세팅은 npx create-expo-app <프로젝트명> --template 명령어를 통해 typescript와 navigation이 가능한 한경을 구성했습니다.
- eSLint & prettier : 코드 스타일을 유지해 가독성을 높이기 위해 사용했습니다.
- jest : 단위 테스트를 도와주는 라이브러리로 에러를 확인해 코드 품질을 높이기 위해 사용했습니다.
